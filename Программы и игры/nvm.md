**Установка:**
```bash
paru -S nvm
```
**Запускаем скрипт для инициализации папок для пользователя:**
```bash
source /usr/share/nvm/init-nvm.sh
```
**Добавляем в при запуске терминала:**
```bash
echo 'source /usr/share/nvm/init-nvm.sh' >> ~/.zshrc
```
**Установка последнего NodeJS:**
```bash
npm install node
```
