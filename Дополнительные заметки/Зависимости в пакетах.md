# Можно отключить проверку зависимостей
## В pacman можно отключить проверку зависимостей при установки/удалении пакетов
Установка без проверки зависимости:
```bash
pacman -Sdd <package>
```
Удаление без проверки зависимости
```bash
pacman -Rdd <package>
```
